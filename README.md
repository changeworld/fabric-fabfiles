# Fabric fabfiles

My fabric files.

## Requirements

* Python 2.7
* Fabric 1.10.1
* Paramiko 1.15.2

## Installation

    $ git clone https://github.com/changeworld/fabric-fabfiles.git

## Usage

### tag

    $ fab tag:[tag_name],[tag_message]
